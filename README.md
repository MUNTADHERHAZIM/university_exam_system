# 🎓 نظام الاختبارات الجامعي
## University Examination System — Django

نظام اختبارات جامعي متكامل مبني بـ Django مع دعم كامل للغة العربية.

---

## 🚀 التشغيل السريع

```bash
cd university_exam_system
chmod +x start.sh
./start.sh
```

ثم افتح: **http://127.0.0.1:8000**

---

## ✨ المميزات الكاملة

### للمشرف / الأستاذ
- ✅ إنشاء الاختبارات وتعديلها وحذفها
- ✅ إضافة **8 أنواع مختلفة** من الأسئلة
- ✅ نشر الاختبار / إيقافه
- ✅ عرض النتائج والإحصائيات
- ✅ **التصحيح اليدوي** للأسئلة المقالية والقصيرة
- ✅ **المراقبة المباشرة** لجلسات الاختبار
- ✅ إنهاء جلسة أي طالب قسراً
- ✅ لوحة التحكم مع الإحصائيات

### للطالب
- ✅ رؤية الاختبارات المتاحة
- ✅ واجهة اختبار احترافية مع عداد الوقت
- ✅ التنقل بين الأسئلة وتمييزها 🚩
- ✅ حفظ الإجابات تلقائياً
- ✅ **التصحيح التلقائي الفوري** عند التسليم
- ✅ عرض النتيجة مع التفاصيل والشرح

### نظام المراقبة والأمان
- 🛡️ رصد تحويل التبويب وإبلاغ المشرف
- 🛡️ منع النسخ واختصارات لوحة المفاتيح

## 🏗️ هيكل المشروع

```
university_exam_system/
├── manage.py
├── requirements.txt
├── start.sh
├── university_exam/          # إعدادات Django
│   ├── settings.py
│   └── urls.py
├── accounts/                 # نظام المستخدمين
│   ├── models.py (Profile)
│   └── views.py
├── exams/                    # نظام الاختبارات
│   ├── models.py             # كل النماذج
│   ├── views.py              # كل الـ Views
│   ├── grading.py            # محرك التصحيح التلقائي
│   └── management/commands/
│       └── seed_data.py      # البيانات التجريبية
├── templates/                # قوالب HTML
│   ├── base.html
│   ├── accounts/
│   └── exams/
└── static/
    ├── css/style.css
    ├── css/exam_take.css
    └── js/main.js
```

## ⚙️ المتطلبات

- Python 3.8+
- pip

جميع المكتبات الأخرى في `requirements.txt`

---

## 🔧 الإعداد اليدوي

```bash
pip install -r requirements.txt
python manage.py makemigrations accounts exams
python manage.py migrate
python manage.py seed_data
python manage.py runserver
```
